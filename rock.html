SUMMARY

The public site of the web application consists of five landing pages for different categories while the admins can manage the songs, albums, and singers in their own dashboards after logging in the application. The front end of the application is built with HTML, CSS, SASS, jQuery as well as the PHP blade templating language, while the back end of the application is built completely with PHP Laravel MVC framework.
